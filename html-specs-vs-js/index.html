HTML5 vs Vanilla JS

I was curious to see how the new HTML5 compared to older javascript solutions

In the javascript solutions, I tried to mimic the behaviour of the HTML5 elements as closely as possible

This Comparision is best viewed in a modern browser

Accordion

HTML5 Accordion

Advantages
  • Easy to implement
  • Works on most browsers
  • You can customize the arrow
  • No special Javascript
  • No special CSS
Disadvantages
  • You can't animate them

Vanilla JS Accordion

    • You can animate it
    • More control over the behaviour
    • Adds more Javascript to the page
    • Adds more HTML to the page
    • Adds more CSS to the page
